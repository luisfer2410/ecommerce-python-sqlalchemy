# 🛒 Tienda de Ropa Fullstack - Python

Este proyecto es una plataforma de e-commerce funcional diseñada para la gestión de ventas, usuarios y catálogo de productos.

## 🚀 Tecnologías utilizadas
* **Backend:** Python con SQLAlchemy (ORM).
* **Frontend:** HTML5, CSS3 y JavaScript.
* **Base de Datos:** SQLite / PostgreSQL.
* **Gestión:** Metodología Kanban (Jira).

## 🛠️ Funcionalidades
- **Módulo de Usuarios:** Registro, login y validación de sesiones.
- **Catálogo Dinámico:** Visualización de productos desde la base de datos.
- **Gestión de Órdenes:** Visualizacion de Carrito de compras y Simulacion de factura.
